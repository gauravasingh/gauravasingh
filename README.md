# Minimal portfolio scaffold

This is a small static portfolio scaffold (HTML, CSS, JS) modeled after a simple personal site design.

Getting started

- Open `index.html` in your browser.
- Or run a simple local server (recommended) and visit http://localhost:8000

PowerShell (Windows) — using Python 3:

```powershell
python -m http.server 8000
```

or with Node (if you have `serve`):

```powershell
npx serve . -l 8000
```

Customization

- Edit `index.html` to replace name, bio, and project links.
- Edit `styles.css` to adjust colors, spacing, and layout.
- Edit `script.js` for interactive behavior.
